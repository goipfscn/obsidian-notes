# AD 预测市场机会挖掘策略

## 触发频率
- 每3小时自动扫描
- 整点30分触发（09:30 / 12:30 / 15:30 / 18:30...）
- Job ID: 8b99137a

## 双轨扫描

### 1. 新市场轨道（优先）
- 上线 < 48小时

### 2. 热门市场轨道
- 成交量：$1K - $50K
- 流动性：> $500

## 过滤条件
- 偏差 < 15% → 跳过
- 流动性 < $500 → 跳过
- 50/50 无信息优势 → 跳过
- 纯加密 Up/Down 循环市场 → 跳过

## 六类市场分类

| 类型 | 描述 |
|------|------|
| 🆕 新市场冷启动 | 上线<48h，价格发现不足，早进吃最大偏差窗口 |
| ⚡ 突发事件狙击 | 外部事件触发，需30分钟内快速判断，标注紧迫程度 |
| ⏰ 临期理财 | 1-7天到期，No吃时间价值，胜率>80% |
| 🔬 基本面套利 | 定价偏离现实，需信息优势，中长期持有 |
| 🎰 彩票仓位 | 5-20¢买入，低概率高赔率，心理上视为已亏损 |
| 👁️ 常识型 | 公众人物固定习惯，市场定价错误，胜率极高 |

## 风险等级

| 等级 | 条件 |
|------|------|
| 🟢 低风险 | 胜率高 + 偏差大 + 流动性好 |
| 🟡 中风险 | 有不确定性 |
| 🔴 高风险/彩票 | 高风险 |

## 信号卡片格式

```
[类型] [风险等级] [偏差%] → 市场名 + 方向 + 价格 + 真实概率 + 结算日期
✅ 正向理由
⚠️ 反向风险
```
