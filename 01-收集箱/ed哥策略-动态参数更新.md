# Ed哥策略 - 动态市场参数更新

## 核心框架（四要素不变）
1. 简单动物题材
2. 主流媒体曝光
3. 一眼看懂，出圈
4. 持续发酵叙事

## 动态参数（定期更新）

### 更新频率
- 每 1-2 周更新一次
- 或者每发现新赛道时更新

### 需要跟踪的数据

#### 1. 市值分布
- 50k-200k 区间币种比例
- 突破 1M 的比例
- 5M+ 的比例

#### 2. 赛道占比
- 动物 IP %
- AI Agent %
- 政治热点 %
- 其他赛道

#### 3. 催化类型效果
- Reuters/主流媒体
- TikTok
- 大 KOL
- 社区自发

#### 4. 最佳参数
- 上车区间
- 目标区间
- 合理盈亏比
