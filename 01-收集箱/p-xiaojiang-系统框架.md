# P小将 画像预测系统 - 框架

## 核心流程

```
新币出现 → 提取特征 → 匹配画像 → 预测买入 → 提前买入 → 卖出获利
```

## 数据输入

| 来源 | 内容 |
|------|------|
| GMGN | 追踪的 CA、交易记录、持仓 |
| Twitter | 讨论的代币、关键词、互动 |
| 链上 | 买入时间、金额、止盈止损 |

## 画像模型

```yaml
P小将:
  - username: 账号
  - chain: 链 (BSC/Solana)
  - preferences:
      市值: [5位, 6位, 7位+]
      叙事: [meme, AI, 链游, VC背景...]
      时机: [上线即买, 等回调, 等Twitter讨论...]
      周期: [短线, 中线, 长线]
      止盈: [2x, 3x, 5x, 10x+]
  - behavior:
      决策来源: [GMGN, Twitter, 自己研究...]
      风险: [低, 中, 高]
```

## 预测方法

### 1. 特征提取
```
新币 → 市值 + 叙事 + Twitter讨论 + 热度
```

### 2. 匹配打分
```
新币特征 vs P小将偏好 = 匹配度
```

### 3. 预测输出
```
匹配度 > 80% → 预测: 会买
匹配度 < 50% → 预测: 不会买
```

## 系统架构

```
┌─────────────────────────────────────┐

               ▼

               │
               ▼
┌─────────────────────────────────────┐
│           输出                        │

└─────────────────────────────────────┘
```
