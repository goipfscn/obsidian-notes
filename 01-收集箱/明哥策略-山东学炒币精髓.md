# 明哥策略 - 山东学炒币精髓

## 核心理念
山东学 = 人情世故 + 揣摩上意 + 把握时机

## 核心观点

### 1. 关系大于能力
- 币圈本质是关系户的市场
- 有关系 = 上币安 = 赚钱
- 没能力 = 再好的项目也上不了

### 2. 领导的话要反着听
- "不要发政治币" = 心里其实想上
- "just meme" = 免责声明
- 模糊语言 = 留后路 + 避免责任

### 3. 过度表达才是合适的
- 请客吃饭：菜刚好吃完 = 招待不周
- 约会：尺寸刚刚好 = 不够好
- 舔老板：过度舔 = 刚刚好

### 4. 以拙胜巧
- 曹植写华丽文章送别 = 被曹操杀了
- 曹丕只哭 = 被立为太子
- pnut 上币安 = 看似随意 实则政治献媚

## 实战应用

### 选币逻辑
1. 看关系：谁上的币安？
2. 看风向：领导最近表达什么倾向？
3. 看时机：什么时候该上车？
4. 看态度：对项目不敬 = 上不了

### 判断标准
| 因素 | 说明 |
|------|------|
| 绑定关系 | 是否和币安/二圣有关系 |
| 社区态度 | 骂币安 = 上不了 |
| 上币时机 | 第一个 = 写进历史 |
| 政治倾向 | 符合当前政治正确 |

### 山东学选币公式
```
有关系 + 不过度表达 + 符合风向 = 大金狗
```
