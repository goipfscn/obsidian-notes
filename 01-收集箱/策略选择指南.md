# 策略选择指南

## 链与策略对应

| 链 | 使用策略 | 核心逻辑 |
|---|---------|----------|
| **Solana** | Ed 策略 | 四要素：大金狗筛选 |
| **BSC** | 明哥策略 | 山东学：关系+风向 |

---

## Solana → Ed 策略

### 适用链
- Solana (SOL)
- 其他海外链

### 核心逻辑
1. 简单动物题材
2. 主流媒体曝光
3. 一眼看懂，出圈
4. 持续发酵叙事

### 数据支持
- GMGN
- Dexscreener
- Twitter 热度

---

## BSC → 明哥策略

### 适用链
- BSC (BNB Chain)
- 币安生态

### 核心逻辑
1. 关系绑定（币安/何一/CZ）
2. 政治风向（平台在推什么）
3. 社区态度（是否对平台恭敬）
4. 以拙胜巧（自上而下）

### 数据来源
- Twitter 搜索
- 何一/CZ 互动
- 社区情绪

---

## 使用示例

### 发现新币 $XXX
1. 判断链：Solana? → 用 Ed 策略
2. 判断链：BSC? → 用明哥策略

### 分析步骤

**Solana (Ed):**
```
1. 检查四要素
2. 看市值/流动性
3. 判断是否符合大金狗模型
```

**BSC (明哥):**
```
1. Twitter 搜索
2. 看何一/CZ 有无互动
3. 是不是"自上而下"
4. 用山东学分析
```
