# 天气交易系统 - 最终方案

## 核心理念

> yuyu: "永远交易的是波动和概率"

核心是找到**模型预测 vs Polymarket定价**的差距。

---

## 正确的推演逻辑

```
1. 获取当天9点的气象数据（气温、湿度、风速、云量、风向）
2. 用ML模型学习 "9点气象 → 当天最高温" 的映射
3. 模型预测当天最高温
4. 对比Polymarket目标温度
5. 计算边缘(edge) = 模型预测概率 - PM定价
6. 边缘 > 阈值 → 下注
```

---

## ML模型

### 特征
- temp_9 (9点气温) - **最重要 92.7%**
- humidity_9 (湿度)
- wind_9 (风速)
- wind_dir_9 (风向)
- cloud_9 (云量)
- pressure_9 (气压)
- month (月份)
- is_calm (静风 <3m/s)
- is_sunny (晴天 <20%云量)
- is_warm_dir (暖风向 135-225°)

### 模型
- GradientBoostingRegressor
- 预测准确率: 平均误差 0.7°C

---

## 交易策略

### 下注条件
- 高温信号: is_sunny + is_calm + is_warm_dir ≥ 2
- 边缘(edge) > 10%

### 盈亏计算
```
边缘 = 模型预测概率 - Polymarket定价
赔率 = 1 / Polymarket定价
盈利 = (赔率 - 1) × 本金 (赢了)
亏损 = 本金 (输了)
```

---

## 统计数据

- 高温信号样本: 445次
- 达标率: ~60%
- 预测误差: 0.7°C
- 边缘>10%时ROI: ~98%

## 🚨 重要实战经验 (来自yuyu D30复盘)

### 2026-03-17 上海案例

**逆温现象**：晚上比白天热！

| 时间 | 操作 | 价格 | 结果 |
|------|------|------|------|
| 10:00 | 买11°C Yes | $8 | 3x出场 |
| 12:00 | 买12°C No | $31 | 翻倍出场 |
| 18:00 | 买11°C No | $85 | 吃到 |
| - | 买13°C Yes | $9→$56 | 5-6x出场 |

### 关键洞察

1. **逆温现象**：下雨+云多+海风 → 晚上升温
   - 特殊天气条件下，晚上可能比白天更热
   - 小白以为太阳下山就结束 ❌

2. **波动交易**：反复交易赚取波动收益
   - 不只是预测温度
   - 在波动中反复做T

3. **Metar播报时间**：23:30
   - 23:30前的Metar属于当天
   - 0:00的Metar属于第二天

4. **博弈区间**
   - 关注概率变化区间(11-13°C)
   - 吃Fomo情绪的钱

5. **交易机会识别**
   - 早上10点：买Yes
   - 中午：买No
   - 傍晚：吃情绪反转

### 更新后的模型特征

新增特征：
- **is_rain**: 是否下雨 (precipitation > 0)
- **is_sea_wind**: 是否海风 (风向180-270°+高湿度)
- **is_cloudy**: 是否阴天 (cloud_cover > 80%)
- **evening_rise_potential**: 晚间升温潜力 (基于逆温条件)
