# P小将 风格预测系统

## 目标
预测 P小将 的买入喜好，他们买入后卖出获利

## 核心逻辑
```
新币特征 → 匹配 P小将 画像 → 提前买入 → P小将买入确认 → 卖出
```

## 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    数据层                               │
├─────────────────────────────────────────────────────────┤
│  链上数据          │       Twitter 数据                  │
│  - 买入记录        │       - 发推内容                   │
│  - 持仓变化        │       - 互动内容                   │
│  - 时间戳          │       - 关键词                     │
└────────┬───────────┴──────────────┬────────────────────┘
         │                          │
         ▼                          ▼
┌─────────────────────────────────────────────────────────┐
│                  特征工程层                            │
├─────────────────────────────────────────────────────────┤
│  币特征标签                                        │
│  - 市值：5位/6位/7位+                                │
│  - 叙事：AI/meme/链游/VC/社区                        │
│  - 热度：Twitter热度/Google趋势                        │
│  - 时间：上线1h/24h/7d                                │
│  - 代码：开源/匿名                                    │
│  - 社区：Discord/TG规模                               │
└─────────────────────────────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────────────────────┐
│                  模型层                                │
├─────────────────────────────────────────────────────────┤
│  P小将 画像库                                        │
│  ┌─────────────┬─────────────┬─────────────┐          │
│  │ @A          │ @B          │ @C          │          │
│  │ 市值:5位    │ 市值:6位    │ 叙事:AI    │          │

│  └─────────────┴─────────────┴─────────────┘          │

│  5. 卖出                                              │
```

## 数据来源

### 链上数据
- Helius API - 交易记录
- Birdeye - 持仓变化
- Dexscreener - 新币提醒

### Twitter 数据
- agent-reach / bird - 读取推文
- 关键词提取

## P小将 画像模板

```yaml
username: "@xxx"
标签:
  - "市值偏好": "5位/6位/7位+"
  - "叙事偏好": "AI/meme/链游/VC/社区"
  - "热度偏好": "高热度/低热度"
  - "买入时机": "上线即买/回调买/稳定买"
  - "持仓周期": "短/中/长"
  - "止盈习惯": "2x/5x/10x/跑路"
典型操作:
  - "2026-01-15": "$XXX 买入 $500"
  - "2026-01-20": "$YYY 卖出 3x"
关键词:
  - "p"
  - "meme"
  - "sunpump"
```
